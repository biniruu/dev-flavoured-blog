a 태그에 `name` 속성을 사용한다면?

<br />

```html
<h2>
  <a name="a1-run-only-for-browser-users" href="#a1-run-only-for-browser-users" class=""></a>
  A.1 - Run Only for browser users
</h2>
```

<br />

사이트 운영에 문제는 없어요. 근데 표준에는 맞지 않아요. w3schools에서는 다음 태그에서만 `name` 속성을 사용하도록 안내하고 있어요.

<br />

**name 태그를 사용할 수 있는 태그 목록**

- button
- fieldset
- form
- iframe
- input
- map
- meta
- object
- output
- param
- select
- textarea
